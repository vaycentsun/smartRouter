## 🚀 快速赞助

运行以下命令显示赞助二维码：

```bash
# 显示二维码图片（如果已配置）
smart-router coffee

# 或使用短命令
smr coffee

# 使用 ASCII 艺术模式（无需图片）
smr coffee --ascii

# 生成指定链接的二维码
smr coffee --link "https://your-sponsor-link"
```

---

## 🎨 自定义您的收款二维码

### 方法 1：替换图片文件（推荐）

1. 准备您的收款二维码图片（支付宝/微信/GitHub Sponsors 等）
2. 将图片保存到以下路径：
   ```
   src/smart_router/assets/coffee_qr.png
   ```
3. 运行 `smr coffee` 即可显示您的二维码

### 方法 2：使用链接生成

```bash
# 使用您的赞助链接生成二维码
smr coffee --link "https://github.com/sponsors/yourname"

# 或使用支付宝/微信收款链接
smr coffee --link "https://qr.alipay.com/xxxxxx"
```
